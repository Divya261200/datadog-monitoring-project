# datadog-monitoring-project

# Cloud Infrastructure Monitoring — AWS + Datadog + Grafana

# Project Overview
  Built an end-to-end cloud monitoring solution on AWS EC2 using Datadog and Grafana for real-time infrastructure observability.

# Tools Used
	∙	AWS EC2
	∙	Datadog
	∙	Grafana
	
# What I Built
	∙	Deployed AWS EC2 instance and installed Datadog agent
	∙	Created dashboards tracking CPU, memory and disk metrics
	∙	Configured High CPU Alert — triggers at 80% threshold
	∙	Set up automated Slack notification workflow for incident response
	∙	Connected Grafana to Datadog for unified visualization
	∙	Configured security group for controlled access

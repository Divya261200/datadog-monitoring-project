# datadog-monitoring-project

# Cloud Infrastructure Monitoring — AWS + Datadog + Grafana

# Project Overview
  Built an end-to-end cloud monitoring solution on AWS EC2 using Datadog and Grafana, including real-time alerting and validation through simulated system load.

# Tools Used
	∙	AWS EC2
	∙	Datadog
	∙	Grafana
	
# What I Built 

1) Deployed AWS EC2 instance and configured Datadog agent for real-time infrastructure monitoring
2) Created dashboards tracking CPU, memory, and disk utilization with threshold-based alerting (CPU > 80%)
3) Implemented automated Slack alerting workflow for proactive incident notification
4) Integrated Grafana with Datadog for unified observability visualization
5) Simulated CPU saturation using controlled processes and validated monitoring accuracy by correlating Datadog metrics with system-level tools (top)
6) Analyzed metric aggregation delay and observed system recovery behavior after terminating load processes
7) Configured AWS security groups for controlled access
